
# React Portfolio & Resume

This is a single-page portfolio and resume application built with React.js and Vite. It features a clean, modern, and responsive design, including sections for profile, experience, education, skills, and contact information.

## Getting Started

### Install dependencies
```
npm install
```

### Start the development server
```
npm run dev
```

## Project Structure
- `src/` — Main source code
- `src/components/` — React components for each section
- `src/App.jsx` — Main app layout

## Customization
Edit the content in the components to update your profile, experience, education, skills, and contact info.
